THU MUC SLIDE MAU
=================

Dat file PPTX vao day de import lam template.

Huong dan:
1. Tai slide tu Canva duoi dang PPTX (File > Tai xuong > PPTX)
2. Dat file .pptx vao thu muc nay
3. Vao /Admin/ImportPptx de chon file va import

Luu y:
- Chi ho tro dinh dang .pptx (Microsoft PowerPoint)
- Moi file se duoc chuyen thanh 1 template co the chinh sua
- Text, anh, vi tri se duoc giu nguyen va co the thay doi

Thu muc nay da san sang. Hay bat dau them slide mau!
